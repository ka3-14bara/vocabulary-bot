Бот получает сообщение, обрабатывает его и записывает в текстовое сообщение, названием которого является id
пользователя. Если такого файла не существует, то при попытке записи он создается автоматически, в ином случае
выдается сообщение об отсутствии такого файла.

Для обработки полученной пары слов используется функция "take_in", в которой сообщение делится на пару EN-RU
путем разделения по ключеваому символу "-", а затем записывается в словарь, в котором английское слово является
ключом, а русское - значением,  и возвращается. 

Для сохранения используется функция "saving". В нем проверяется присутствует ли файл по заданному пути, затем записывает
пару или создает файл, а затем записывает.

---------!!! ВНИМАНИЕ !!!-----------
Функции удаления и сортировки с выдачей по заданной букве не введены в эксплуатацию в основной файл bot_body.

Функция удаления работает следующим образом: с файла пользователя считываются все пары и записываются в "word_pairs", 
файл полностью очищается, затем происходит поиск в списке нужного слова. Если такое слово находится в нем, то оно удаляется,
затем вызывается функция сохранения. Поиск чувствителен к регистру.

Поиск и выдача по первой заданной букве осуществялется так же, как и при поиске в функции удалении, только производится 
сравнение первого символа, так же используется словарь.
